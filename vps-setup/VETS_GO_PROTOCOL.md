# VETS GO Warning Protocol

Per SOP and Bible directive: Any item, script, deployment, or system state that requires explicit human authorization from VETS before live activation MUST use the following warning block format. 

This block must be bold, use sirens, and be highly visible (not fine print).

```markdown
🚨 **VETS GO REQUIRED** 🚨
**The following items remain on HOLD and require explicit authorization to proceed:**
- [Item 1]
- [Item 2]
**DO NOT ACTIVATE LIVE WITHOUT EXPLICIT "VETS GO" COMMAND.**
```
