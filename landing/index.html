Features

Everything you need. Nothing you don't.

Homebrew meets Docker Desktop meets an AI-native App Store. One installer to rule them all.

🧙

Step-by-Step Wizard

Guided setup from system check to running services. No YAML required. No terminal gymnastics.

🤖

AI-First

Install OpenClaw, Hermes, or NanoClaw with one click. Add local LLMs via Ollama or bring your own API key.

🐳

Docker-Powered

Everything runs in containers. Docker Compose generated dynamically from your selections. Clean, portable, reproducible.

🔒

Secure by Default

Auto-generated strong credentials. Traefik + Let's Encrypt SSL. Secrets never leave your machine.

📦

In-App Store

Install, update, and remove services after setup. Browse official and community packages. One-click management.

🌐

Any Network

Localhost dev, LAN home server, or internet-facing with auto SSL. OpenTang adapts to your setup.

How It Works

From zero to running in 9 steps

OpenTang walks you through everything. System checks, package selection, credentials, deployment — all handled.

1

Welcome

Launch OpenTang and hit "Begin Setup"

2

System Check

Auto-detects OS, Docker, RAM, disk. Fixes missing deps

3

Choose Edition

NanoClaw, Hermes, or OpenClaw — pick your tier

4

LLM Config

Install Ollama locally or enter your cloud API key

5

Select Packages

Toggle services on/off with live resource estimates

6

Network Setup

Local, LAN, or internet-facing with domain + SSL

7

Security

Auto-generated credentials. Copy, save, or vault them

8

Install

Watch each service deploy with progress and log viewer

9

Done 🎉
